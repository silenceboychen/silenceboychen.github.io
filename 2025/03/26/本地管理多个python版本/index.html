工具 适用系统 特点
pyenv macOS/Linux 轻量级，纯命令行操作
conda 跨平台 集成包管理，适合科学计算
Docker 跨平台 完全环境隔离，但需学习容器技术
手动管理 所有系统 灵活性高，但维护成本较高
