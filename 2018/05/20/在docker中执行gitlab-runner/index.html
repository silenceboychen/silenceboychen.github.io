环境:ubuntu 16.04 LTS
目的：使用Docker安装和配置GitLab Runner，搭建GitLab CI持续集成环境。

安装gitlab-runner

参数说明：

  • -d: 设置容器后台运行
  • –name：容器名称
  • -restart always：每次启动容器就重启 gitlab-runner
  • -v: 共享目录挂载

安装好后，执行$ docker ps 查看容器是否运行。
